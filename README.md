# WMS Backend - Grupo 4 (AS/RS Automatizado)

Servidor de orquestación central para el sistema AS/RS, construido con FastAPI, PostgreSQL y MQTT.

## Arquitectura de Ramas (Git Flow)
⚠️ **Regla de Oro:** Nadie programa directamente en `main`.
1. Crea tu rama: `git checkout -b feat/tu-tarea`
2. Trabaja y haz commits.
3. Sube tu rama y pide integración (Merge).

## Instalación rápida para el equipo
1. Clona el repositorio: `git clone [URL-DE-TU-REPOSITORIO]`
2. Crea el entorno virtual: `python -m venv venv`
3. Activa el entorno (Windows): `.\venv\Scripts\activate`
4. Instala las dependencias: `pip install -r requirements.txt`
